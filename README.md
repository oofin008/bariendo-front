## Getting Started
First, setup firebase credential in `src/core/data/auth/firebaseAuth.ts`
Then, install dependencies and run the development server:

```bash
npm install
npm run dev
```

#### Routes
Base route: localhost:3000
- /signin
- /signup
- /dashboard
